## **七彩虹CVN B660I GAMING FROZEN V20+12400+5500xt**

## 配置

| 配置   | 型号                              |
| ------ | --------------------------------- |
| CPU    | intel i5 12400                    |
| 主板   | 七彩虹CVN B660I GAMING FROZEN V20 |
| 显卡   | 5500xt                            |
| 内存   | 英睿达DDR4 3200 8Gx2              |
| 网卡   | ax201                             |
| 硬盘   | 三星980 1T                        |
| OC版本 | 0.8.3                             |
| macOS  | macOS Monterey 12.5.1             |
| 机型   | MacPro7.1                         |

> BIOS设置
>
> 1. 关闭CSM（启动模式仅UEFI）
> 2. 关闭快速开机 Fast Boot
> 3. 关闭Intel VT-D
